# Incognito Chat App 

This repository contains the source code for the official website of the Incognito Chat App, a privacy-focused social networking platform. The application is designed to help people stay connected while ensuring their conversations remain private and are automatically erased after they end.

## Features

*   **Privacy**: All conversations are designed with user privacy configuration.
*   **Auto-Erase Conversations**: Chats are automatically deleted after the conversation ends.
*   **Multi-Conversation Support**: Easily host and switch between multiple conversations at the same time.
*   **Built-in Translations**: The app supports translations to communicate with people from around the world.

## Repository Overview

This repository hosts the static web pages for [incognitochatapp.com](https://incognitochatapp.com).

### Key Pages

*   `index.html`: The main landing page for the application.
*   `Api.html`: Provides a link to the Incognito Chat API documentation.
*   `Jobs.html`: A page with a form for submitting job applications.
*   `Support.html`: Contains contact information for customer support.
*   `PrivacyPolicy.html`: The official privacy policy for the service.
*   `EULA.html`: The End-User License Agreement.
*   `404.html`: Custom page for handling "Not Found" errors.
